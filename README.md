# vajraDemo

Steps to Run Project

1)First Clone the Project

  Type below command in the terminal:-
  
  git clone https://github.com/sawan456/vajraDemo
  
2)Install Node Modules

  Type below command in the terminal:-
  
  npm install
  
3)Now Run the Project

  Type below command in the terminal:-
  
  npm run dev
